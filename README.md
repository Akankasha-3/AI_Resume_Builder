## 🚀 Getting Started

### 📦 Backend Setup (Node.js / Express)

1. Navigate to the root directory:
   ```bash
   cd RESUME BUILDER

2. Install dependencies::
   ```bash
   npm install
   
3. Start the Server:
    ```bash
    npm start

💻 Frontend Setup (Vite + React)
1. Navigate to the frontend folder:
   ```bash
   cd my-app

3. Install dependencies:
   ```bash
   npm install
5. Start the development server:
   ```bash
   npm run dev
